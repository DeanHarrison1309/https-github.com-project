1.1 IT in Modern Business

IT plays a major role in modern business, vastly increasing the speed and efficiency of a business. Data storage and Information management are vital cogs in the running of a business. The difference in terms of efficiency between storing data on computers, which can be accessed in a matter of seconds compared to retrieving paper documents kept in filing cabinets (not to mention the floor space saved), is astronomic. Information management (stock maintenance, sales information, customer relationships) are much easier to keep track of through IT. Keeping such data organised will help to improve the productivity of a business, thus increasing profits. Communication with customers is another area where time and speed of delivery via IT bears no comparison to traditional methods.

1.2 Methods, Skills and Resources needed to complete digital projects.

There are 3 essential resources that you need in order to develop a nice website. Javascript: Javascript (JS) is a scripting languages, primarily used on the Web. It is used to enhance HTML pages and is commonly found embedded in HTML code. HTML: Hypertext markup language (HTML) is the major markup language used to display Web pages on the Internet. In other words, Web pages are composed of HTML, which is used to display text, images or other resources through a Web browser. CSS: Cascading Style Sheets (CSS) is a standard (or language) that describes the formatting of markup language pages. CSS enables developers to separate content and visual elements for greater page control and flexibility. A CSS file is normally attached to an HTML file by means of a link in the HTML file. (techopedia.com/dictionary) In simple terms, HTML is the structure for your website, Javascript contains and adds information to the site, and CSS is used to style your website. 1.3 Plan and Carry Out Tasks using IT

A digital project comprises the planning, preparation, and production that goes into creating electronic publications. Possible electronic publications include: Web sites, electronic books, digital images, etc. (https://cdrh.unl.edu/articles/basicguide/digitalproject) Let’s focus on a website. Planning: First of all, let us decided on a theme for the website. What is it about? What should it contain? What is the goal/purpose of this site? What age bracket/audience is it aimed at? What will attract this audience to the site? How can the site differ to and improve upon currently existing sites of a similar theme? Preparation: Let us now do some research. We need to gather as much information as we can about the topic. Collect as many relevant facts and figures as possible. Find at least 5 websites similar to that which we have in mind and compare these sites. What elements are common to each site? Look for something unique about a site. Production: Production of our website will begin with building the structure (HTML). Think about how we want the site to look like? How many pages do we want? How much information per page? What should be on the homepage? The structure complete, we can then incorporate our Javascript and finally decorate the site using CSS.
